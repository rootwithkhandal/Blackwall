Frequently Asked Questions

Everything you need to know about Solfugu.

What is Solana Fugu? +
Solana Fugu (Solfugu) is an advanced multi-agent algorithmic trading platform built specifically for the Solana blockchain, offering high-speed execution and AI-driven strategies.
How does the multi-agent system work? +
The system decouples data aggregation, analysis, and execution into separate specialized agents. This ensures that market monitoring never blocks trade execution.
Is my private key secure? +
Yes, private keys never leave your local environment. The execution agents run within your own infrastructure and only sign transactions locally.
Can I write custom strategies? +
Absolutely. We provide a comprehensive SDK in Rust and Python to develop and deploy custom Analysis agents to the network.
