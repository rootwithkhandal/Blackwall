Architecture

BlackWall relies on decoupled components to deliver a scalable, fault-tolerant network monitor and firewall engine.

1. Packet Capture (Scapy)

Daemon threads silently capture live network traffic using Scapy, extracting essential layers (IP, TCP, UDP) without blocking Streamlit's UI loop.

2. Machine Learning Detector

Utilizing scikit-learn's Isolation Forest, BlackWall builds a baseline for 5 minutes. Any deviations from normal rate or port heuristics trigger automated mitigation.

3. Immutable JSONL Ledger

To provide a clear audit trail, every auto-ban and allow rule is appended to a local JSONL ledger file.

4. Multi-SIEM Forwarding

Bans and drops are instantly sent to Splunk HEC, Wazuh, Discord, and Slack via an asynchronous queue system, alerting security engineers the moment an incident occurs.
