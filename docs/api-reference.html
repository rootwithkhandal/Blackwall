API Reference

Welcome to the BlackWall REST API documentation. The local API runs on port 8000 and provides programmable access to the firewall engine and the immutable ledger.

GET /rules

Returns the currently active user-defined firewall rules.

curl http://localhost:8000/rules

GET /banned

Returns a list of IP addresses that have been automatically banned by the ML anomaly detector.

curl http://localhost:8000/banned

GET /stats

Returns real-time packet statistics including total allowed, dropped, and banned connections.

curl http://localhost:8000/stats

GET /ledger

Returns the recent cryptographic blocks from the immutable ledger representing firewall decisions.

curl http://localhost:8000/ledger?limit=10
